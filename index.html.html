⚙️ Configuración

Para enviar los gastos a Google Sheets necesitás pegar la URL de tu Google Apps Script web app. Te explico cómo crearla abajo.

¿Cómo creo el script? Ver instrucciones

📋 Instrucciones

Seguí estos pasos para conectar con Google Sheets:

1. Abrí Google Sheets y creá una hoja nueva llamada "Gastos"

2. En la fila 1 poné estos encabezados:
Fecha · Concepto · Monto · Caja · Moneda

3. Andá a Extensiones → Apps Script

4. Borrá lo que hay y pegá el código de abajo

5. Guardá y hacé click en Implementar → Nueva implementación

6. Tipo: App web · Acceso: Cualquier usuario

7. Copiá la URL y pegala acá

function doPost(e) { var sheet = SpreadsheetApp .getActiveSpreadsheet() .getSheetByName('Gastos'); var data = JSON.parse(e.postData.contents); sheet.appendRow([ data.fecha, data.concepto, data.monto, data.caja, data.moneda ]); return ContentService .createTextOutput(JSON.stringify({ok:true})) .setMimeType(ContentService.MimeType.JSON); }
